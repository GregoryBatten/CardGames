Project Description

This is an all-in-one casino card game application to demonstrate abstraction, encapsulation, inheritance, and 
polymorphism. The application GUI implements Swing from the Java library. Future plans include games with machine 
learning for intelligent AI opponents, database management for login credentials and player metadata, and computer 
vision for text and card recognition. More details at the bottom of this page.

Blackjack Rules

- Every round will use 1 deck, freshly shuffled.
- The dealer must hit on a soft 17.
- Cards with the same rank can be split for a maximum of 4 total hands.
- Face cards can split with other face cards but not with a 10. 
- A blackjack can only be the first 2 cards dealt. Splitting a hand cannot result in a blackjack.
- When the dealer's face up card is an ace, the user can pay up to half of their original bet for insurance.
- A blackjack pays 3:2 of the original bet. A standard win pays 1:1 of the original bet.

Future Plans

First, I am going to create an SQL database to manage multiple user accounts, including their credentials and game-
related metadata. There will be a login screen that presets the userPlayer for the rest of the application. Then I 
plan to develope more complicated card games, starting with Texas Hold'em. This game will allow me to create 
intelligent, machine learning AI opponents that calculate their moves based off win probabilities and trends in the
user's activity. There is also room to increment the game with online multiplayer, either as a PC, mobile, or 
browser application. Opponents can be AI or human in a lobby with private room networking and chat capabilities.
Finally, every feature developed so far can be reused for a card dueling game, such as Pokemon or Magic the
Gathering. Developement should include computer vision to insert cards that a user has at home into a database for 
their virtual deck, and image/text processing to determine that card's capabilities in the game. Animations and 
sound presentation can be improved and stylized.
